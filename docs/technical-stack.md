# Technical Stack

## Frontend

- Next.js (version 15 with TurboPack, written in TypeScript), ShadCN UI, next-themes, TailwindCSS, Axios, Framer Motion.

## Backend

- NestJS (written in TypeScript), TypeORM, PostgreSQL, JWT.

## DevOps

- Docker, Free Tier Services.

## Testing

- Jest, Cypress, ESLint.
